# Hand Gesture Recognition using SVM

## Objective
Develop a hand gesture recognition model that identifies and classifies different hand gestures from image data.

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn

## Machine Learning Algorithm
- Support Vector Machine (SVM)

## Process
1. Image preprocessing
2. Feature extraction
3. Model training
4. Gesture classification

## Output
Recognized and classified different hand gestures with accuracy score.
